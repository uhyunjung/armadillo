## 📖 프로젝트 이름 : 아르마딜로 대시
아르마딜로 대시는 1명의 보스 플레이어와 4명의 아르마딜로 플레이어가 방송국 점거 및 생존을 위해 대결하는 탄막 게임입니다.
유니티, 포톤, 깃허브로 제작하였고, 수정본 및 최종본을 업로드하였습니다.


## 👨‍👩‍👦‍👦 팀원 소개
|이름|역할|
|------|---|
|허선|PM, 기획, 그래픽, 코더(4번 탄막)|
|유현정|코더(탄막 버튼, 1번 탄막, 룸, 서버 동기화)|
|임아현|코더(플레이어, 6번 탄막, 서버 동기화)|
|조서원|그래픽|
|차주영|코더(2번, 3번, 5번 탄막)|
|최한비|코더(플레이어, 로비)|
